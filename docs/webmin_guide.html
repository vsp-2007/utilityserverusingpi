Webmin Guide

Simplified Server Management for your Raspberry Pi

🚀 What is Webmin?

Webmin is an advanced web-based interface for system administration. It simplifies the process of managing your Linux server by allowing you to perform complex tasks through an easy-to-use GUI, automatically updating all required configuration files for you.

Key Benefits:

  • Syntax Protection: Prevents errors and warns you before potentially dangerous actions.
  • Remote Access: Manage your server from any device on your network via a browser.
  • Modular Design: Over 100 modules to manage everything from Users to Firewalls.
  • Seamless Mixing: Recognizes manual configuration changes made via the terminal.

🔑 Permissions Simplified

Permissions control who can read, write, or enter a folder. Instead of using complex letters, Webmin uses simple codes.

700 Locked Down Only the owner has access. Maximum privacy.
770 Team Access The owner and members of the group can access.
777 Public Everyone on the network can see and edit.

📂 Samba Windows File Sharing

Follow these steps to create folders that show up in Windows File Explorer or Mac Finder.

1. Private Personal Share Recommendation

Access: Only you.

Samba ➝ Create a new file share
  • Share Name: private_vault
  • Permissions: 700
  • Security ➝ Valid Users: Your username
  • File Permissions ➝ New File Mode: 600 (Only owner reads/writes)

2. Family/Team Share Shared

Access: Everyone in a specific group.

Samba ➝ Create a new file share
  • Share Name: family_photos
  • Create with Group: family
  • Permissions: 770
  • Security ➝ Valid Users: @family

3. Public Guest Share Open

Access: Anyone on WiFi (No password).

Samba ➝ Create a new file share
  • Share Name: guest_files
  • Owner / Group: nobody / nogroup
  • Permissions: 777
  • Security ➝ Guest Access: Yes

📊 Bandwidth Monitoring

Advanced Info

Monitor real-time traffic to see which services or devices are using your internet.

Quick Setup

Networking ➝ Bandwidth Monitoring
  • Select Setup limits on selected interfaces.
  • Select your interface (eth0 for Ethernet, wlan0 for WiFi).
  • Click Setup Now to start logging.
  • Go to Daily Traffic to view usage stats.
⚠️ IMPORTANT: This feature logs every packet header to a database. On a Raspberry Pi, this can slow down your network and fill up your SD card if left on forever. Use it for debugging, then turn it off.
